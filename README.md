## 📌 Overview

Exploratory data analysis (EDA) and predictive modelling on an insurance dataset. Uncovers key drivers of insurance claims and costs through statistical analysis, visualisation, and regression modelling.

**Tech Stack:** `Python` `Pandas` `Seaborn` `Scikit-learn` `EDA` `Data Science`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Insurance_analysis.git
cd Insurance_analysis
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
